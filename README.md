# 종합 알고리즘 시뮬레이터

## 프로젝트 개요
**종합 알고리즘 시뮬레이터**는 알고리즘 학습을 위한 웹 기반 시각화 플랫폼입니다.  
정렬, 그래프, 트리, 동적 계획법(DP) 등의 알고리즘을 단계별로 시각화하고,  
사용자가 직접 입력하거나 자동 생성한 테스트케이스를 실험하며 결과를 기록할 수 있습니다.

## 핵심 기능

### 1. 시뮬레이터 (단계별 애니메이션)
- 정렬, 그래프(탐색/최단경로/MST), 트리(순회/DFS·BFS), DP(LCS/Knapsack) 알고리즘 시각화
- 각 단계에서 내부 자료구조(큐/스택/우선순위큐, 거리배열, parent, DP 테이블) 실시간 표시
- 일시정지, 배속, 이전/다음 단계 이동 기능 제공

#### 알고리즘 범위
**정렬**  
- Bubble, Insertion, Selection, Merge, Quick, Heap Sort  
- 비교/교환 횟수 및 시간 측정, 하이라이트 표시  

**그래프**  
- 탐색: BFS, DFS (큐/스택 상태 실시간 표시)  
- 최단경로: Dijkstra, Bellman-Ford, Floyd-Warshall  
- MST: Kruskal, Prim (Union-Find 시각화)

**트리**  
- 순회: Pre/In/Post/Level-order  
- BST 삽입/삭제, 힙(상향/하향 힙화) 시각화  

**DP (동적 계획법)**  
- 0/1 Knapsack, LCS, Coin Change  
- DP 테이블 채우기 과정 및 선택 추적 경로 표시

### 2. 테스트케이스 자동 생성기
- 문제 유형별 파라미터를 설정하여 자동으로 입력 데이터 생성
  - 그래프: 정점 수, 간선 밀도, 가중치 범위, 연결성 보장 등
  - 정렬: 배열 길이, 분포(균일, 역정렬 등)
  - DP: Knapsack, LCS 등 각 문제 유형별 파라미터 지원
- 난이도 프리셋 (쉬움 / 보통 / 어려움)
- 엣지케이스 전용 생성 옵션
- JSON/CSV 내보내기 및 불러오기 지원

### 3. 그래프·트리 그림판 에디터
- 마우스로 **정점 추가 / 삭제 / 이동**
- **간선 드래그 연결**, **가중치 편집**, **방향 그래프** 지원
- **자유 배치 / 자동 레이아웃(dagre, cola 등)** 토글 가능
- **부분 실행**: 현재 캔버스 상태에서 알고리즘 실행 (예: Dijkstra)
- 실행 결과 **경로 / 트리 하이라이트 애니메이션** 표시

### 4. 학습 기록 탭
- 오늘 푼 문제, 배운 점, 스크린샷 및 데이터 첨부 가능
- **태그** 기능 지원 (#그래프, #DP 등)
- **목록 / 검색 / 정렬** 기능 제공
- **Markdown 형식 작성 지원**
